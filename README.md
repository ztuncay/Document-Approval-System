# Document Approval System

This project is a role-based document approval system developed with ASP.NET Core and MSSQL.

## Features
- Role-based authorization
- Document upload and management
- Approval / rejection workflow
- Secure user authentication
- MSSQL database integration

## Technologies
- ASP.NET Core
- C#
- MSSQL
- Entity Framework Core

## Purpose
This project demonstrates a simple workflow system where users can upload documents and authorized users can approve or reject them.
